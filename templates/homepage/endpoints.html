Instructivo para usar los Endpoints de la API

Endpoints Disponibles

Servicios

  • Lista de servicios: GET /api/servicios/
  • Detalle de servicio: GET /api/servicios/<id>

Clientes

  • Lista de clientes: GET /api/clientes/
  • Detalle de cliente: GET /api/clientes/<id>

Empleados

  • Lista de empleados: GET /api/empleados/
  • Detalle de empleado: GET /api/empleados/<id>

Coordinadores

  • Lista de coordinadores: GET /api/coordinadores/
  • Detalle de coordinador: GET /api/coordinadores/<id>

Notas Importantes

  1. Todos los endpoints responden solo a solicitudes GET
  2. Los parámetros deben enviarse en la URL
  3. Las respuestas siempre son en formato JSON
  4. Para endpoints de detalle, reemplazar <id> por el ID numérico
